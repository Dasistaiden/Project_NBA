# 01. 價值主張與需求總覽 — Yahoo NBA Fantasy 選秀模擬器

## 1. 專案背景與動機

Yahoo Fantasy NBA 選秀時，玩家需要在極短時間內綜合大量資訊做出選擇：球員的場上數據表現、健康狀況、球隊角色定位、未來走勢、甚至是心理素質，這些資訊分散在不同來源，難以在選秀當下快速整合比較。本專案的目的是打造一個選秀模擬器，把這些分散的判斷因素整合成單一、可比較、可調整的分數與介面，幫助自己在 2025-2026 賽季的 Yahoo Fantasy 選秀中做出更有依據的選擇。

## 2. 核心價值主張

> 將球員的量化場上數據，與健康度、球隊角色、新秀潛力、心態等非量化因素，整合成一個可依 Yahoo 玩法自訂加權的 Fantasy Point 分數，並依五大位置分類呈現，讓選秀決策從「憑印象」變成「有數據佐證」。

## 3. 目標使用者

- 使用者本人，用於準備並實際操作 2025-2026 賽季的 Yahoo Fantasy NBA 選秀。

## 4. 功能範圍

### 4.1 資料來源
- 使用 `nba_api`、NBA 官網或其他可靠來源，取得 2025-2026 年度已登錄 NBA 球員的綜合統計數據。

### 4.2 計算方式（Fantasy Point）
- 各項數據（得分、籃板、助攻、抄截、阻攻、失誤、命中率等）依 Yahoo 當年玩法給予不同加權。
- 加權為可調整變數，需提供合理預設值，使用者可依當年聯盟規則自行調整。
- 各項數據 × 對應權重加總，即為該球員的 Fantasy Point。

### 4.3 數值外因素考量
除了場上數據，模擬器需一併評估以下四項因素，作為 Fantasy Point 之外的補充判斷依據：

1. **健康度**：球員過去 n 年的出勤狀況與傷病紀錄，並評估下賽季的受傷風險指數。
2. **球隊角色定位**：先發或替補、是否適配球隊戰術、屬於主要進攻發起者還是藍領工兵型角色。
3. **新秀分析**：新秀無 NBA 賽場數據，需以 NCAA（或其他聯賽）比賽紀錄等替代資料評估其潛力。
4. **心態**：近年分析師重視、但非傳統數據無法呈現的特質，例如球員展現積極進取或消極擺爛的傾向。

### 4.4 呈現方式
- 建立 Streamlit 介面。
- 依 PG / SG / SF / PF / C 五大位置分類，列出各位置的適配人選。
- 每位球員需附上前一年度的平均數據，作為排序與選擇的佐證。

## 5. 成功標準

- 使用者能在介面上依五大位置查看候選球員的排序清單。
- 每位候選球員的排序結果附有前一年平均數據與加權後 Fantasy Point，足以支持人工判斷。
- 加權參數可調整，並能即時反映在排序結果上。
